# Android Google Play Store Submission Guide

This guide provides step-by-step instructions for building and submitting the X39 Matrix app to Google Play Console.

## Prerequisites

Before building the Android App Bundle (AAB), ensure you have:

1. **Android Studio** (latest stable version) or **Android SDK Command Line Tools**
2. **Java Development Kit (JDK)** 11 or higher
3. **Node.js** and **pnpm** installed
4. **Signing keystore** for release builds (see Keystore Setup below)
5. **Google Play Console** account with app created
